# Learning Roadmap

The journey is divided into structured phases covering all important DSA topics:

## Phase 0: Basics

* Problem-solving fundamentals
* Arrays & simple logic
* Time & Space Complexity

## Phase 1: Arrays, Strings & HashMaps

* Arrays
* Strings
* Hashing techniques
* Prefix sums

## Phase 2: Searching, Sorting & Sliding Window

* Binary Search
* Sorting algorithms
* Two pointers
* Sliding Window patterns

## Phase 3: Linked Lists, Stacks & Queues

* Singly & Doubly Linked Lists
* Stack problems
* Queue implementations
* Monotonic Stack/Queue

## Phase 4: Recursion, Backtracking & Trees

* Recursion fundamentals
* Backtracking problems
* Binary Trees
* BSTs & Tree Traversals

## Phase 5: Greedy, Intervals & Bit Manipulation

* Greedy algorithms
* Interval problems
* Bitwise operations

## Phase 6: Graphs & Union-Find

* BFS & DFS
* Shortest paths
* Topological sorting
* Disjoint Set Union (DSU)

## Phase 7: Dynamic Programming & Advanced Topics

* Dynamic Programming
* Trie
* Heap/Priority Queue
* Segment Trees
* Advanced interview problems
